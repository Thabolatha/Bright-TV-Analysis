# 📺 Bright-TV-Analysis

## Project Overview
This repository contains a comprehensive analysis of BrightTV's user behaviour 
and viewership trends. The project explores user demographics, consumption patterns, 
peak viewing times, top channels, and geographic distribution across South Africa. 
The insights derived from this analysis are intended to inform strategic business 
decisions to grow BrightTV's user base and increase platform engagement.

## 🔍 Key Areas of Analysis

- **User Demographics** — Age group distribution, gender breakdown, and provincial reach
- **Platform Health** — Active vs inactive users and overall retention rate
- **Viewing Behaviour** — Sessions by day, time of day analysis, and peak engagement windows
- **Content Performance** — Top channels by views and average watch time per channel
- **Geographic Trends** — User distribution across all South African provinces
- **Viewership Over Time** — Monthly trend analysis and spike investigation

---

## 💡 Key Insights

- **91% active user rate** out of 10,989 total users showing strong platform retention
- **Gauteng dominates** with ~3,700 users followed by Western Cape at ~1,850
- **25-34 Young Adults** make up 39% of the user base — the core audience
- **Friday afternoons** are the peak viewing window across all days and times
- **ICC Cricket World Cup 2011** records the highest average watch time at 16+ minutes
- **SuperSport Live Events** leads in total views confirming sport as the strongest content pillar

in Databricks SQL to create a unified dataset for analysis.

## 🚀 Recommendations Summary

- Secure live sport and cricket broadcasting rights to retain highly engaged users
- Expand music and entertainment catalogue to diversify beyond sport content
- Launch targeted marketing in underperforming provinces — Eastern Cape, North West & Northern Cape
- Introduce localized content in indigenous languages — Zulu, Xhosa & Sotho
- Redesign onboarding to capture complete user profiles for better personalisation
- Schedule premium content on Friday afternoons and weekends to maximise viewership
